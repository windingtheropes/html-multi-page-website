blogger

Blogger is my latest project, a customizable blog engine originally written in Javascript, and later rewritten in Typescript. Through this project, I've learned to write type safe object oriented code, through a custom-made database, utilizing file I/O and caching to serve blog data efficiently.

purge

Purge is a simple command line application that can mass-delete files by matching patterns, following customizable parameters. It was originally written in Javascript running through Node.js, although this solution was very bulky and slow, contradicting the quick and convenient nature of the application. I decided to rewrite it in Rust as one of my first projects in Rust, and I quickly realized the benefits of Rust, and was forced to write safer code. Linked above is the purge-rust rewrite, the old version written in javascript can be found here.

  • Javascript

  • Rust

keeb

Keeb is one of my more advanced projects, my foray into working with HID through Rust, and using the C programming language. Keeb works by contacting a mechanical keyboard running QMK, allowing for control of keymappings and raw input without flashing firmware every time. Keeb has been a great learning experience, requiring various rewrites, and lots of thinking. Although the project has become stale, it has served useful for learning more advanced Rust features, and working with HID devices.

  • C

  • QMK
